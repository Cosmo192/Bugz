# Bug Bounty Finder

Bug Bounty Finder is a web app that scans GitHub repositories and dependency files for known vulnerabilities using OSV.dev.

## For Users

Use the deployed app URL in your browser.


## For Maintainers

Use this section when you want to update and redeploy the app.

1. Make code changes locally.
2. Test locally:

```bash
npm install
npm run dev
npm run build
```

3. Commit and push to GitHub:

```bash
git add .
git commit -m "Update app"
git push
```
